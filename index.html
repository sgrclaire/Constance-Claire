« interMatch » : rencontres par goûts culturels

Web documentaire réalisé dans le cadre du cours « Document numérique ». Objectif : montrer qu’on peut faire du matching à partir de Spotify, Babelio, Letterboxd et d’un petit test de personnalité (mode clair = ressemblant, mode sombre = opposé).

1. Présentation rapide du projet

L’idée est de proposer un site de rencontre qui ne se base pas uniquement sur la photo, mais sur les activités culturelles réelles des utilisateurs. On part de leurs playlists Spotify, de leurs lectures Babelio et de leurs films vus sur Letterboxd, puis on ajoute une couche de personnalité pour décider d’afficher plutôt des profils « ressemblants » ou au contraire « complémentaires ».

Le document que tu es en train de lire est la version HTML5 demandée par l’enseignant : il contient un sommaire, des sections distinctes, un glossaire, et une bibliographie interactive.

2. Glossaire

API
Interface permettant de récupérer les données d’une autre application (ici Spotify, Babelio, Letterboxd).
Matching
Action de rapprocher deux profils selon des critères (goûts, personnalité).
Web documentaire
Page web structurée mêlant texte, médias et navigation interne.
